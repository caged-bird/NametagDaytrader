# nametag

### Getting Started

#### If you use your own system,

```
npm install
npm run demo:local
```
then

check [http://localhost:1337](http://localhost:3000/)

#### If you use Vagrant,

on Root directory

```
vagrant up
vagrant ssh
```

on Vagrant root

```
cd codes
npm install
npm run demo:local
```

then

check [http://127.0.0.1:1337](http://127.0.0.1:1337/)
